Family Car Wash – Premium Door‑step Service
A small, fully‑featured Next.js 13 + App Router portfolio site that demonstrates:

Server‑rendered pages with
Navbar, footer and layout
Email‑contact form (front‑end form + /api/contact handler)
Simple next‑auth login (MongoDB + Credentials provider)
Tailwind CSS for a dark‑theme UI
Framer Motion for entrance animations
TypeScript throughout the stack
Built for Vercel but works on any Node 14+ environment.
